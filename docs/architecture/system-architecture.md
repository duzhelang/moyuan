# 古今诗话——墨渊 系统架构文档

## 项目概述

**项目名称**：古今诗话——墨渊（SC\_MoYuan2）

**项目定位**：中国古典诗词文化交流平台，集诗词欣赏、诗人介绍、论坛互动、AI辅助创作于一体的综合性文化网站。

**核心价值**：传承中华诗词文化，提供现代化的诗词学习与交流体验。

### 模块划分

#### 1. 首页模块

- **功能**：轮播图展示、导航菜单、诗人推荐、作品展示、分类精赏
- **特点**：响应式布局、动态加载、SEO友好
- **状态**：前端已实现（views/home/index.vue）

#### 2. 诗词模块

- **功能**：诗词浏览、按朝代/流派/诗人分类、诗词详情、注释赏析
- **数据结构**：诗词表、诗人表、朝代表、流派表
- **状态**：前端已实现（views/poem/list.vue, detail.vue）+ 后端 API（backend PoemController）

#### 3. 论坛模块

- **功能**：用户发帖、评论互动、精选推荐、投稿上传
- **特点**：富文本编辑、图片上传、实时评论
- **状态**：前端已实现（views/forum/list.vue, detail.vue）+ 后端 API（sc-moyuan-backend ForumController）

#### 4. 用户模块

- **功能**：用户注册、登录认证、个人中心、收藏管理
- **安全**：JWT认证、密码加密、会话管理
- **状态**：前端已实现（views/user/login.vue, register.vue, profile.vue）+ 后端 API（sc-moyuan-backend AuthController）

#### 5. AI模块

- **功能**：AI诗词问答、智能分析、诗词推荐
- **技术**：调用大语言模型API（智谱AI、DeepSeek、Kimi、MiMo）
- **状态**：已实现（前端首页AI问答组件 + 后端API（sc-moyuan-backend AiController））

#### 6. 搜索模块

- **功能**：全文搜索、诗人搜索、诗词搜索、帖子搜索
- **技术**：MySQL全文索引 + LIKE模糊查询
- **状态**：已实现跨模块搜索（`GET /api/search?keyword=` 支持诗词+诗人+帖子搜索）

#### 7. 管理模块（新增）

- **功能**：系统管理、用户管理、内容管理、数据统计、操作日志
- **架构**：前后端分离，管理后台独立布局
- **后端**：AdminController（backend模块）
- **前端**：AdminLayout + 管理页面组件（含ECharts图表）
- **权限**：管理员专属，需要管理员角色认证
- **状态**：已实现

#### 8. 文件模块（新增）

- **功能**：文件上传、图片管理
- **技术**：本地文件存储 + 静态资源映射
- **后端**：FileController + WebMvcConfig
- **前端**：ImageUpload通用组件
- **状态**：已实现

#### 9. 浏览历史模块（新增）

- **功能**：记录用户浏览历史、查看历史记录、清空历史
- **数据结构**：user_history表
- **后端**：HistoryController
- **状态**：已实现

### 后端技术栈

> **说明**：项目存在两套后端代码——`backend/`（基础版）和 `sc-moyuan-backend/`（增强版）。基础版提供诗词相关基础 API，增强版在基础版基础上增加了 JWT 认证、Knife4j 文档、Druid 连接池、AOP/Validation 等能力，并实现了用户认证和论坛功能。
>
> **Java 版本**：pom.xml 中 `java.version` 设置为 23，但 maven-compiler-plugin 的 source/target 设为 17，实际开发编译使用 **Java 17 兼容模式**。

## 数据流架构

### 请求流程

```
用户请求 → Nginx(反向代理) → Vue前端 → API请求 → Spring Boot后端 → MySQL/Redis → 返回数据 → 前端渲染
```

### 认证流程

```
用户登录 → 后端验证 → 生成JWT → 返回Token → 前端存储 → 后续请求携带Token → 后端验证Token
```

## 部署架构

### 开发环境

- 前端：Vite Dev Server (localhost:5173)
- 后端：Spring Boot (localhost:8080)

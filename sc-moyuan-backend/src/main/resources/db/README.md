## 使用方法

### 方式一：命令行初始化

```bash
# 进入脚本目录
cd sc-moyuan-backend/src/main/resources/db

# 执行初始化（会提示输入密码）
mysql -u root -p < init.sql
```

### 方式二：MySQL客户端导入

1. 打开MySQL客户端（如Navicat、DBeaver、MySQL Workbench）
2. 连接到MySQL服务器
3. 执行 `init.sql` 文件内容

### 方式三：Spring Boot自动初始化

在 `application.yml` 中配置：

```yaml
spring:
  sql:
    init:
      mode: always
      schema-locations: classpath:db/init.sql
```

> **注意**：此方式仅建议在开发环境使用，生产环境请手动执行初始化。

## 前置条件

- MySQL 8.0+
- 足够的数据库创建权限
- 字符集支持 utf8mb4

## 注意事项

1. **幂等性**：所有建表使用 `IF NOT EXISTS`，数据使用 `INSERT IGNORE`，可重复执行
2. **外键约束**：init.sql 中未包含外键约束，应用层保证数据一致性
3. **密码加密**：用户密码使用 BCrypt 加密存储
4. **API密钥**：AI模型的 `api_key` 为占位符，需替换为真实密钥
5. **逻辑删除**：大部分表支持逻辑删除（`deleted` 字段）

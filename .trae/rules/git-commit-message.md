## 提交信息格式

### 基本结构
```
<类型>: <描述>
```

### 类型前缀
| 类型 | 适用场景 |
|------|---------|
| `[文档]` | 文档、README、技能文件变更 |
| `[数据库]` | SQL脚本、数据库结构变更 |
| `[算法]` | Python算法、模型文件变更 |
| `[前端]` | Vue组件、页面、样式变更 |
| `[后端]` | Java后端、API接口变更 |
| `[修复]` | Bug修复 |
| `[优化]` | 性能优化、重构 |
| `[配置]` | 配置文件变更 |

### 示例
```
[文档] 更新README和技能文档
[数据库] 更新SQL脚本
[算法] 更新Python算法代码
[前端] 更新Vue前端代码
[后端] 更新Java后端代码
[优化] 页面拆分重置
```

### 规范要求
- 描述使用简体中文
- 控制在 50 字以内
- 清晰说明变更内容，避免模糊描述
